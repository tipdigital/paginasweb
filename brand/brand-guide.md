# 🎨 Brand Guide - TipDigital

Bem-vindo ao guia de identidade visual do projeto. Use essas diretrizes para manter a consistência em todas as páginas do funil.

---

## 🎨 Paleta de Cores e Tokens

| Categoria | Token | Valor | Uso |
| :--- | :--- | :--- | :--- |
| **Backdrop** | `--bg-color` | `#0c0814` | Cor de fundo geral |
| **Accent Primary** | `--accent-color` | `#00ff88` | Botões de ação (CTA), Destaques |
| **Secondary/Pulse** | `--secondary-color` | `#7000ff` | Efeitos de glow, hovers e títulos |
| **Static Text (Heading)** | `--text-primary` | `#ffffff` | Títulos e Textos Principais |
| **Static Text (Sub)** | `--text-secondary` | `#b0a8c4` | Subtítulos e Textos Auxiliares |

---

## ✒️ Tipografia e Fontes

*   **Heading (Títulos)**: `Outfit`, sans-serif
    *   *Uso*: `<h1>` até `<h4>`.
    *   *Estilo*: Negritos fortes (`font-weight: 800`), espaçamento ligeiramente negativo.
*   **Body (Textos)**: `Inter`, sans-serif
    *   *Uso*: Parágrafos, listas, legendas.
    *   *Estilo*: Muito legível, altura de linha padrão (`line-height: 1.6`).

---

## 🔬 Componentes Globais (Aesthetic)

*   **Cards**: Devem usar `backdrop-filter: blur(12px)` (Glassmorphism) com bordas finas semi-transparentes brancas (`rgba(255,255,255,0.06)`).
*   **Glows**: Efeitos de brilho roxo/verde no fundo atraem os olhos para os principais CTAs.
